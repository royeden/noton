# Noton

A minimal chr editor, written in ANSI C.

## Build

To build noton, you must have [SDL2](https://wiki.libsdl.org/).

```
sudo apt-get install libsdl2-dev
```

## Controls

### General

- `BACKTICK` Cleanup Brush
- `1-7` Patterns
- `TAB` Cycle between colors
- `H` Toggle Guides
- `Z` Decr. Brush Size
- `X` Incr. Brush Size
- `N` Clear

### Paint

- `mouse1` Paint
- `mouse1+mouse2` Erase
